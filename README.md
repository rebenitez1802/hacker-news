# hacker-news

##To run the app:
	1. Clone the repo 
	2. run `npm install`
	2. run `node index.js`
	3. open a browser in http://localhost:3333/

## To force the load of data:
	with the server running open http://localhost:3333/load

## the name of the database in mongo is `test_database` and the collection `articles`
